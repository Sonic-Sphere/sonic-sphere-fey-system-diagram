## Executive Summary

The rack shown in the screenshots is almost certainly built around a TASCAM ML-32D 32-channel analog/Dante converter. The likely playback path is:

`MacBook Air M2 / Dante source -> USB-C multiport adapter Ethernet -> TP-Link 5-port switch -> TASCAM ML-32D -> DB25 analog output fanouts -> Dayton MA1240a top amp + two Dayton MA1260 lower amps -> 30x Focal 100 OD8 passive outdoor speakers`.

The powered subwoofer is a Devine Onyx 18SXA. The TASCAM feeds one RCA channel into a Devine DI-003 box, and the DI-003 output goes to the subwoofer left-channel input. The cleanest 30.1 mapping still reserves ML-32D analog output 31 for the sub line input and leaves analog output 32 spare, but the exact TASCAM source output should be verified at the rack.

The Bluetooth-Dante adapter is confirmed as an Audinate Dante AVIO Bluetooth Adapter (`ADP-BT-AU-2X1`). It connects to the Dante switch by RJ45 and needs PoE from the switch or from a PoE injector. The TP-Link / Omada injector is now identified as `TL-POE150S` / `POE150S`. A Ubiquiti `GPH240-125G` one-port injector and a Ubiquiti EdgePoint R6 (`EP-R6`) are also present; their exact roles in the active Dante/site network still need rack tracing.

## Network / Dante Layer

```text
                         Bluetooth
                    phone / tablet / PC
                             ~~~
                              |
                              v
                +-----------------------------+
                | Bluetooth-Dante adapter     |
                | Dante AVIO Bluetooth        |
                | RJ45 Dante + PoE            |
                +---------------+-------------+
                                |
                                | Cat5e+ STP, Dante, PoE if AVIO
                                v
+-------------------+     +-----------------------------+     +--------------------+
| MacBook Air M2    |     | TP-Link 5-port switch        |     | TASCAM ML-32D      |
| DVS / app /       +---->| 4 ports currently occupied   +---->| PRIMARY RJ45       |
| Dante Controller  |     | standard Ethernet            |     | Dante network      |
| via USB-C adapter |     |                             |     |                    |
+-------------------+     +-----------------------------+     +--------------------+

MacBook adapter:
- USB-C multiport adapter with 3x USB-A, 1x Ethernet, 1x HDMI, and 1x USB-C power input.
- Ethernet from this adapter is the wired MacBook path into the Dante switch.

Additional network hardware to trace:
- TP-Link TL-POE150S / Omada POE150S: one-port IEEE 802.3af PoE injector; identify the powered device.
- Ubiquiti GPH240-125G: one-port 24 V PoE injector; identify the powered device.
- Ubiquiti EdgePoint R6 (`EP-R6`): outdoor EdgeMAX router/control point; verify whether it is upstream of the Dante switch, substituting for a switch/router, or isolated from the Dante audio network.

TASCAM ML-32D SECONDARY RJ45:
- Leave disconnected in a simple single-switch Dante network, or
- Use only for Dante redundancy with a completely separate secondary switch/network.
- Do not connect Primary and Secondary to the same redundant network.
```

## Proposed Logical Channel Assignment

This is a clean logical map, not proof of the actual on-site patching.

| Sonic Sphere channel | TASCAM ML-32D analog out | Amp | Amp channel | Destination |
| --- | ---: | --- | ---: | --- |
| Speaker 1-12 | 1-12 | MA1240a top unit | 1-12 | Focal 100 OD8 speakers 1-12 |
| Speaker 13-24 | 13-24 | MA1260 middle unit | 1-12 | Focal 100 OD8 speakers 13-24 |
| Speaker 25-30 | 25-30 | MA1260 bottom unit | 1-6 | Focal 100 OD8 speakers 25-30 |
| Sub / LFE | 31 | Devine DI-003 -> Devine Onyx 18SXA | Left input | TASCAM RCA channel into DI-003, then DI output to sub left channel |
| Spare | 32 | n/a | n/a | Spare ML-32D output |
| Spare amp channels | n/a | MA1260 bottom unit | 7-12 | Unused or future expansion |

Physical DB25 grouping is by blocks of eight:

| ML-32D output connector | Channels |
| --- | --- |
| Analog outputs 1-8 | 1-8 |
| Analog outputs 9-16 | 9-16 |
| Analog outputs 17-24 | 17-24 |
| Analog outputs 25-32 | 25-32 |

Because the amp channels are grouped in twelves and the TASCAM connectors are grouped in eights, at least one fanout block likely crosses an amplifier boundary unless there is an intermediate patch panel or custom loom.

## Field Verification Checklist

1. Photograph the rear of the TASCAM and confirm the exact model text is `ML-32D`.
2. Confirm which ML-32D RJ45 is patched: Primary should go to the switch in the simple network.
3. Photograph the switch model and confirm whether any ports are PoE.
4. Trace the TP-Link TL-POE150S / Omada POE150S injector and record what powered device is on its PoE output.
5. Trace the Ubiquiti GPH240-125G injector and record what powered device is on its PoE output.
6. Trace the Ubiquiti EdgePoint R6 (`EP-R6`) ports and power input; confirm whether it is upstream of the Dante switch, part of the Dante path, or separate site-network hardware.
7. Confirm the Dante AVIO Bluetooth device name in Dante Controller.
8. Trace the Devine Onyx 18SXA sub cable: source device/channel 31 to the Devine DI-003 RCA input, then DI-003 output to the subwoofer left-channel input.
9. Read labels on all DB25 fanouts. Record exact model numbers: `DTM`, `DTF`, `DTP`, custom, DB25-to-RCA, etc.
10. Confirm whether the DB25-to-XLR cable is plugged into ML-32D analog inputs or outputs.
11. In Dante Controller, export or screenshot subscriptions: MacBook/DVS/Bluetooth transmitters to ML-32D receive channels.
12. Confirm amp input switches are set to `LINE IN` for discrete 30-channel playback, not `BUS 1` or `BUS 2`.
13. Label MA1240a / MA1260 speaker output terminals by Sonic Sphere speaker number.

## Source Notes

- TASCAM states the ML-32D has 32 analog line inputs and 32 analog line outputs using D-Sub connectors, plus Dante Primary/Secondary connectors.
- TASCAM's manual says the Primary connector should be used for the Dante network, and the unit is not compatible with 100 Mbps switching hubs.
- User rack note identifies the amplifier stack as one Dayton Audio MA1240a on top and two Dayton Audio MA1260 units below it.
- Dayton Audio states the MA1240a has 12 channels, individual RCA line inputs, bus inputs/outputs, Phoenix-style speaker outputs, and 40 W RMS/channel at 8 ohms.
- Focal states the 100 OD8 is an 8 ohm passive outdoor loudspeaker with IP66 certification and recommended amp power of 20-150 W.
- Audinate states the Dante AVIO Bluetooth adapter is RJ45, PoE powered, 48 kHz, 24-bit, and `ADP-BT-AU-2X1`.
- User rack note confirms the Dante AVIO Bluetooth adapter is present.
- User rack note identifies the network switch as a TP-Link 5-port switch with four occupied ports.
- User update identifies the TP-Link / Omada PoE injector as `TL-POE150S` / `POE150S`, with unknown powered device.
- TP-Link states the POE150S / TL-POE150S is IEEE 802.3af compliant, supports gigabit speed, and provides 15.4 W max 48 VDC output.
- User rack note identifies Ubiquiti `GPH240-125G` as a one-port PoE injector, with unknown connected device.
- User update identifies a Ubiquiti EdgePoint R6 (`EP-R6`) as present.
- Ubiquiti states the EdgePoint R6 has five 10/100/1000 RJ45 ports, one 1 Gbps SFP port, 24 V passive PoE/DC input options, and 24 V passive PoE output on `eth0`-`eth4`.
- User screenshot identifies the computer as a MacBook Air M2 (2022), Apple M2, 8 GB memory, running macOS Tahoe 26.5.
- User rack note identifies the subwoofer as a Devine Onyx 18SXA fed through a Devine DI-003 box: TASCAM RCA into DI-003, then output to the subwoofer left-channel input.
