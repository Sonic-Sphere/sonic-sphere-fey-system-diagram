# Fey Sonic Sphere System Description

[Open the current version](https://sonic-sphere.github.io/sonic-sphere-fey-system-diagram/#/diagram)

This document is an interactive system diagram for the Chateau du Fey Sonic Sphere audio and network installation. It maps the playback source, Dante interface, network and power hardware, amplifier channels, DI paths, and speaker-zone wiring so the installation can be reviewed as a complete signal-flow reference.
