# Product Manuals

This folder is the single local collection point for manufacturer manuals, quick-start guides, installation guides, owner guides, and product-guide-style documents used by the Sonic Sphere Fey system diagram.

Datasheets and pure specification sheets remain in `../specs/` unless they are the closest available manual-style source for a product.
